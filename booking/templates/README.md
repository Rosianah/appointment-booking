Project specs:

Frameworks used:
1. python3
2. Python Django
3. DB: SQLite

prerequisites

1. VS Code
2. install SQLite viewer extension for Vs Code

To run

1. run "pip install -r requirements.txt"
2. Migrations:  "python manage.py makemigrations" and "python manage.py migrate"
3. Run: python manage.py runserver
